# 🏠 Rental Management MVP

> A modern rental system for managing rooms, users, and listings efficiently.

 

![Banner](https://dummyimage.com/1200x300/0d1117/ffffff&text=Rental+Management+MVP)

 

## 🚀 Overview

Rental Management MVP is a simple yet structured system designed to manage rental listings, user interactions, and basic property operations. Built with a focus on usability, clarity, and Agile workflow.

 

## ✨ Highlights

- 🏡 Room listing & management system  
- 🔐 User authentication system  
- 📊 Agile-based development workflow  
- 🧾 Jira + WBS project tracking  
- ⚡ Clean and responsive UI  

 

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript / React  
- Backend: Node.js / Express.js  
- Database: MongoDB  
- Tools: Jira, Git, GitHub, WBS, Gantt Chart  

 

## 📊 Project Management

Managed using Agile methodology with sprint-based planning. Tasks were tracked using Jira boards and structured using Work Breakdown Structure (WBS).
